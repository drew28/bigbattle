About this game

Motivation

Drooza's Big Battle uses NES series space shooter game named Gradius as motivation. Since there are few flash space shooter flash games on-line this would not be a novel idea but a path not ventured often.

Screen Shots

Game Engine

The game engine is an updated version from a flash game involving many bugs and a net called InsectCollector. Choosing to use this game engine was a decision to evolve the engine; paving a way for other games to be built on this platform.
